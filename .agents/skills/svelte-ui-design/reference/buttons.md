# Buttons

Provides customizable button styles with various sizes and types.

## Basic Implementation

Three button variations:

- Icon-only buttons using `btn-icon preset-filled`
- Standard buttons with `btn preset-filled`
- Buttons combining text and icons

## Preset Styles

Provides full support of Presets with three style variants:

- **Filled**: `preset-filled-[color]-500`
- **Tonal**: `preset-tonal-[color]`
- **Outlined**: `preset-outlined-[color]-500`

Available color options: primary, secondary, tertiary, success, warning, error, and surface.

## Size Options

Three predefined sizes:

- `btn-sm` (Small)
- `btn-base` (Base)
- `btn-lg` (Large)

## States

### Disabled

Apply the native `disabled` attribute to `<button>` elements for disabled styling.

### Button Groups

Group related buttons using the `btn-group` class with optional preset styling. Common use case: monthly selection interface with active state toggling.
