# Cookbook

A collection of recipes for crafting interface features that utilize Skeleton primitives.

## What's This?

These recipes, based on design system recipe concepts, help augment and expand the Skeleton design system and component library.

## Available Recipes

### Blocks/Notifications

- **Alerts** - Alert blocks and notifications

### Blocks/Navigation

- **Breadcrumbs** - Navigation breadcrumbs
- **Stepper** - Step-by-step navigation
- **Table of Contents** - Document navigation

### Blocks/Machine Learning

- **Chat** - Chat interface blocks

### Native/Programmatic

- **Clipboard API** - Programmatic clipboard access

### Native/Popover

- **Dialog Element** - Native dialog elements

### Theme/Guides

- **Dynamic Theme Loading** - Runtime theme switching
- **Light Switch** - Dark mode toggle

### Integration/Popovers/Svelte

- **Floating UI Attachments** - Popover positioning

### Blocks/Layouts/Images

- **Image Layouts** - Responsive image layouts
- **Scroll Containers** - Scrollable image containers

### Blocks/Branding

- **Logo Clouds** - Brand logo displays

### Native/Effects

- **SVG Filters** - SVG filter effects

## Usage

Each recipe provides detailed documentation for implementing specific interface features using Skeleton's design system and components.
