# Chips

Provides a robust set of interactive chip styles.

## Component Types

Three chip variants:

- Icon-only chips using `chip-icon` class
- Text chips using `chip` class
- Combined text and icon chips

## Styling Options

Chips support extensive customization through preset classes:

- **Filled variants**: primary, secondary, tertiary, success, warning, error, surface
- **Tonal variants** for each color
- **Outlined variants**
- **Gradient backgrounds** combining multiple colors

## Interactive States

### Disabled State

Add the `disabled` attribute to button elements to create non-interactive chips.

### Selection Pattern

Implement toggle functionality where chips dynamically switch between:

- `preset-filled` (selected)
- `preset-tonal` (unselected)

Based on user interaction.
